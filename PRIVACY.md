# Privacy Policy

Better Rhythia Downloader does not collect, store, sell, or transmit user data.

The extension runs only on `rhythia.com` map listing pages. It requests Rhythia's public map listing API so it can add direct download buttons to visible map cards.

The extension does not:

- collect personal information
- collect browsing history
- collect cookies, authentication headers, request bodies, or private API payloads
- send data to external servers other than Rhythia's public map listing API
- use analytics
- execute remotely hosted code

All extension code runs locally in the browser.
